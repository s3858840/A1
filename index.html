Home Automation Using
Raspberry-Pi

Overview

This is an alternative and cost-effective Home Automation solution using Raspberry-Pi compared to the existing brands in the markets which are expensive. This will include controlling the Air Condition unit, Lights, TV etc., using the mobile phone. Event the color of lights can be changed. It also switches the light ON and OFF based on the sunlight or available room light which can save electricity. The AC can be turned on before you reach home so that the room will be cooled down.

Motivation

Currently there are Home Automation brands available in the market. But these costs around $3000 and above for the system. By using Raspberry-Pi and 3rd party sensors this cost can be reduced to $500.

Description

The Project aims to replace existing home automation processors with Raspberry-Pi which is more powerful that any of the existing home automation processors. Adding 3rd party control software helps to reduce the cost. On a later stage additional system integration will be possible like background music in each room, even the whole control of electricity.

Tools and Technologies

Hardware required includes, Raspberry-Pi, 3rd party sensors like, temperature sensor, light sensor, RGBW lights etc.

Skills Required

The Programming skill will be required to program Raspberry-Pi. Integration with mobile phone will require to skills to develop a mobile app as well.

Outcome

A Home Automation System with Raspberry-Pi which is cost effective and more powerful than the existing systems
